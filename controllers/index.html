Welcome to bazaar api

Below are some notes on endpoints and routes




Users routes

GET users - list all users

https://bazaar-server.herokuapp.com/api/users

POST users/signup - create new user

https://bazaar-server.herokuapp.com/api/users/signup

Following are required

  1. first_name
  2. last_name
  3. email
  4. password
  5. dob
  6. image (as a file)

POST users/login - authenticates and returns a user object

https://bazaar-server.herokuapp.com/api/users/login

Following are required

  1. email
  2. password
  3. image (as a file and optional)

PUT user - updates user by id

https://bazaar-server.herokuapp.com/api/users/update/:id

DELELTE user - deletes user by id

https://bazaar-server.herokuapp.com/api/users/delete/:id


Sections routes

GET sections - list all sections

https://bazaar-server.herokuapp.com/api/sections

POST sections/create - create new section

https://bazaar-server.herokuapp.com/api/sections/create

Following are required

  1. title
  2. image(as a file)

GET sections/title/:section_title - gets the section object by it's title

https://bazaar-server.herokuapp.com/api/sections/title/:section_title

replace the :section_title in the link with current section title

PUT section - updates section by id

https://bazaar-server.herokuapp.com/api/sections/update/:id

DELELTE section - deletes section by id

https://bazaar-server.herokuapp.com/api/sections/delete/:id


Stalls routes

GET stalls - list all stalls

https://bazaar-server.herokuapp.com/api/stalls

POST stalls/create - create new stall

https://bazaar-server.herokuapp.com/api/stalls/create

Following are required

  1. name (name of the stall)
  2. user (add user id here from signed in user object)
  3. section (add section id here from current section object)
  4. image (as a file)

GET stalls/user/:user_id - list all stalls by user

https://bazaar-server.herokuapp.com/api/stalls/user/:user_id

replace the :user_id in the link with current user object id

GET stalls/section/:section_id - list all stalls by section

https://bazaar-server.herokuapp.com/api/stalls/section/:section_id

replace the :section_id in the link with current section object id

PUT stall - updates stall by id

https://bazaar-server.herokuapp.com/api/stalls/update/:id

DELELTE stall - deletes stall by id

https://bazaar-server.herokuapp.com/api/stalls/delete/:id


Products routes

https://bazaar-server.herokuapp.com/api/products

https://bazaar-server.herokuapp.com/api/products/create

  2. description (name of the product)
  3. user (add user id here from signed in user object)

GET products/user/:user_id - list all products by user

https://bazaar-server.herokuapp.com/api/products/user/:user_id

replace the :user_id in the link with current user object id

GET products/section/:section_id - list all products by section

https://bazaar-server.herokuapp.com/api/products/section/:section_id

replace the :section_id in the link with current section object id

GET products/stall/:stall_id - list all products by stall

https://bazaar-server.herokuapp.com/api/products/stall/:stall_id

replace the :section_id in the link with current section object id

PUT products - updates section by id

DELELTE product - deletes product by id
